# Provider Setup

Your Little Helper supports:

- Local models (Ollama)
- OpenAI (API key)
- Anthropic (API key)
- Google Gemini (API key)

## Local (Recommended)

Local runs on your computer and keeps your data on your device.

1. Install Ollama
2. Start Ollama
3. In Settings → AI Model, pick "Local"

## Cloud providers

Cloud providers require an API key.

In Settings → AI Model → Add API keys:

- OpenAI: paste your OpenAI API key
- Anthropic: paste your Anthropic API key
- Gemini: paste your Gemini API key

Keys are saved locally on your device.
