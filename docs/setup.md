# Setup

Your Little Helper is a desktop assistant that can:

- Find files
- Help fix problems
- Do web research (optional)
- Build projects using Spec Kit (optional)

## Install

Download the latest release from GitHub and run the installer for your platform.

## First run

1. Open the app
2. Pick the folders you want it to work in
3. Choose an AI model in Settings
